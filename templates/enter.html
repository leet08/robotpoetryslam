What is Robot Poetry Slam?


Each player receives a prompt to enter a word or phrase. With that word, the game generates a line of the rhyme, and the player must then write a sentence to complete the rhyme. Once the full rhyme is complete, players will view all the rhyme videos and vote on which one is best.
The player with the most points wins!

How to play?


One person should be designated as Moderator and create a new game by clicking Create New Game. They will receive a Game ID. Each player can then enter the Game ID and create a new username, then click Enter Game. Once the Moderator confirms that all players have joined, they will instruct all players to click the given links to advance to the Writing Round. Following that, players will go into the Voting Round, and finally see the Results.
